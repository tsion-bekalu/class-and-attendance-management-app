# Class and Attendance Management App

This project is a mobile-based class attendance system designed for campus use. It allows instructors tocreate, manage classes and conduct attendance using a secure QR-based session system. Students can mark their attendance only during active sessions, ensuring that only those present in class are recorded. The system also supports class scheduling and announcements to improve communication between instructors and students.
