# Class and Attendance Management App

This project is a mobile-based class attendance system designed for campus use. It allows instructors to create and manage classes, handle student join requests, and conduct attendance using a secure QR-based session system. Students can join classes using a class code, view their weekly timetable, and mark attendance only during active sessions, ensuring that only those present in class are recorded. The system also includes announcements to support communication between instructors and students, and provides attendance summaries that help instructors track student participation and identify low attendance.
